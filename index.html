The Ecological Registry:

Registered Ecological Gardens.

A registry layer for gardens: baseline records, change over time, and optional verification pathways — designed so private land can become legible, connected ecological infrastructure.

Start a conversation View a sample registered garden

A registry operating in two directions.

The work begins in real gardens: reading a site, shaping land, building soil, and planting for long time. From there, the work extends outward: measurement, registry, incentives — the infrastructure that allows ecological gardening to be recognised, valued, and repeated. :contentReference[oaicite:1]{index=1}

The Ecological Registry:

Baseline → change over time.

Each registered garden has a public profile (curated story + images) and an optional private layer (address-level detail, notes, records). The core is the change log: baseline first, then improvements.

The System:

Gardens need infrastructure around them.

Individual gardens matter — but systemic change requires classification, measurement, registry, incentives, and aggregation: the layers that make private ecological work visible and durable. :contentReference[oaicite:2]{index=2}
